## 选定设计方案：Cyberpunk Terminal Elegance

### 设计运动
**Cyberpunk Terminal Elegance** - 融合黑客电影美学、现代毛玻璃效果和科技感的终端风格

### 核心原则
1. **深色极简主义**：深色背景（近黑色 `#0a0e27`）作为主体，减少视觉噪音
2. **荧光对比**：使用霓虹绿（`#00ff88`）和青蓝（`#00d4ff`）作为交互和强调元素
3. **毛玻璃层级**：通过 `backdrop-filter: blur()` 创建浮动感，强化 UI 深度
4. **单色线条**：细致的边框和分割线（`rgba(255,255,255,0.1)`）营造精致感

### 色彩哲学
- **主背景**：`#0a0e27`（深空蓝黑）- 深邃、专业、不疲劳
- **次背景**：`#1a1f3a`（深紫蓝）- 毛玻璃容器的基础
- **强调色**：`#00ff88`（荧光绿）- 成功状态、可交互元素、重点信息
- **辅助强调**：`#00d4ff`（霓虹青）- 链接、悬停状态、次级交互
- **文本主色**：`#e8f0ff`（冷白）- 高对比度，易读性强
- **文本次色**：`#a8b8d8`（冷灰）- 次要信息、禁用状态

### 布局范式
- **左侧导航 + 主内容**：参考现有应用的侧边栏结构
- **浮动卡片**：使用毛玻璃效果的卡片悬浮于背景之上
- **垂直节奏**：充分的纵向间距，营造呼吸感
- **网格对齐**：8px 基础网格，确保精确对齐

### 标志性元素
1. **终端光标动画**：闪烁的竖线光标，暗示"输入就绪"
2. **代码块样式**：使用等宽字体展示城市列表、快速查询等
3. **毛玻璃浮窗**：所有交互容器都采用 `backdrop-filter: blur(10px)` + 半透明背景

### 交互哲学
- **即时反馈**：悬停时改变颜色和模糊度，点击时有缩放反馈
- **层级感**：通过模糊和透明度表现前后关系
- **平滑过渡**：所有颜色变化使用 `transition: all 0.3s cubic-bezier(0.23, 1, 0.32, 1)`

### 动画指南
- **按钮按下**：`transform: scale(0.95)` + `0.16s ease-out`
- **卡片进入**：从 `opacity: 0; transform: translateY(8px)` 到完全显示，`0.3s ease-out`
- **光标闪烁**：`animation: blink 1s infinite`，在终端输入框中
- **悬停效果**：背景模糊度从 `blur(10px)` 增加到 `blur(12px)`，文本颜色切换到强调色

### 排版系统
- **标题字体**：`JetBrains Mono` 或 `IBM Plex Mono`（等宽，科技感）
- **正文字体**：`Inter` 或 `Segoe UI`（清晰易读）
- **标题层级**：
  - H1: `32px / 1.2` 加粗，荧光绿
  - H2: `24px / 1.3` 中等，冷白
  - H3: `18px / 1.4` 正常，冷灰
  - Body: `14px / 1.6` 正常，冷白
  - Caption: `12px / 1.5` 正常，冷灰

## 实现细节

### CSS 变量定义
```css
--bg-primary: #0a0e27;
--bg-secondary: #1a1f3a;
--bg-tertiary: #242d4a;
--accent-green: #00ff88;
--accent-cyan: #00d4ff;
--text-primary: #e8f0ff;
--text-secondary: #a8b8d8;
--border-light: rgba(255, 255, 255, 0.1);
--border-medium: rgba(255, 255, 255, 0.2);
```

### 毛玻璃容器
```css
.glassmorphism {
  background: rgba(26, 31, 58, 0.4);
  backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.1);
  border-radius: 12px;
}
```

### 交互状态
- **Hover**：背景亮度 +5%，模糊度 +2px，光标变绿
- **Active**：缩放 0.95，背景亮度 +10%
- **Disabled**：透明度 50%，文本色变灰

## 页面结构

### 登录页面
- 左侧：终端代码块展示（README 内容）
- 右侧：登录表单（毛玻璃卡片）
- 交互：输入框获焦时显示荧光绿光标

### 首页 / 仪表板
- 顶部：导航栏（毛玻璃，固定）
- 左侧：快速查询面板（城市列表、筛选）
- 中央：主内容区（旅游建议卡片网格）
- 右侧：实时信息面板（天气、交通等）

### 组件库
- Button：荧光绿边框 + 透明背景，悬停时填充
- Input：毛玻璃背景，焦点时边框变青
- Card：毛玻璃容器，阴影柔和
- Badge：荧光绿或青蓝标签，圆角
